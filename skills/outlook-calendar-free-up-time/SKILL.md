# Outlook Calendar Free Up Time

Use calendar evidence before proposing moves.

## Workflow

1. List events in the requested window.
2. Identify tentative, free, or low-priority holds separately from true busy
   meetings.
3. Propose one to three concrete options.
4. Ask for confirmation before moving or canceling anything.
